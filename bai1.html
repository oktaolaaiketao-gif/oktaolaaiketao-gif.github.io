Hướng Dẫn Cơ Bản Về HTML và CSS

Tìm hiểu những kiến thức nền tảng để xây dựng trang web đầu tiên của bạn.

1. HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc và nội dung cho các trang web. HTML không phải là một ngôn ngữ lập trình, nó chỉ định nghĩa các thành phần như tiêu đề, đoạn văn, hình ảnh và liên kết.

Mỗi trang HTML được xây dựng từ các thẻ (tags). Ví dụ: <h1> cho tiêu đề lớn, <p> cho đoạn văn, và <a> cho liên kết.

2. Các Thẻ HTML Cơ Bản

Cấu trúc cơ bản của một trang HTML 
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tiêu đề trang</title>
</head>
<body>
    <!-- Nội dung của trang web -->
</body>
</html>
Các thẻ HTML thông dụng
  • <h1> đến <h6>: Các thẻ tiêu đề.
  • <p>: Đoạn văn bản.
  • <a>: Liên kết (hyperlink).
  • <img>: Hình ảnh. (Lưu ý: Luôn thêm thuộc tính alt để đảm bảo tính truy cập.)
  • <ul>, <ol>, <li>: Danh sách.
  • <div> và <span>: Các thẻ chứa nội dung.

3. CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ được dùng để tạo kiểu cho các thành phần HTML. Nó giúp trang web của bạn trở nên đẹp mắt và có bố cục hợp lý hơn. Với CSS, bạn có thể thay đổi màu sắc, phông chữ, kích thước, và vị trí của các phần tử.

Có ba cách để thêm CSS vào trang web:

  • Inline CSS: Viết trực tiếp vào thẻ HTML bằng thuộc tính style.
  • Internal CSS: Viết trong thẻ <style> ở phần <head>.
  • External CSS: Viết trong một file .css riêng biệt và liên kết đến nó bằng thẻ <link>.

4. Ví dụ về CSS

Dưới đây là một ví dụ về cách áp dụng CSS để thay đổi màu chữ và phông chữ.

Ví dụ Internal CSS 
<head>
    <style>
        h1 {
            color: #007bff;
            font-family: Arial, sans-serif;
        }

        p {
            font-size: 16px;
        }
    </style>
</head>
<body>
    <h1>Tiêu đề trang</h1>
    <p>Đây là một đoạn văn bản được tạo kiểu bằng CSS.</p>
</body>
